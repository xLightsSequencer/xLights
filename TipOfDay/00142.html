Filter The Model List By Regex

The Layout tab now has a filter box that narrows the model list using a regular expression.

Type a pattern into the filter and the model list shrinks to just the models whose names match. Because it accepts a regex you can do more than a simple text search, for example matching several name prefixes at once or models that end in a number.

On a big display with hundreds of models this is a fast way to jump to the ones you care about instead of scrolling through the whole tree.
