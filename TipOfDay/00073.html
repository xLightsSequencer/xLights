Model Pane Count Badges

The Model pane now shows small badges with counts for Faces, States, SubModels, and Nodes on each model.

The numbers appear right alongside the model so you can see at a glance how it is set up without having to open it. A quick scan tells you which models have face or state definitions and which submodels exist.

This is useful when you are working with a large display and trying to remember which props you have already configured, or when you are checking that an imported model came in with everything it should have.
