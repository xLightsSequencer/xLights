Search The Controller Visualizer

The Controller Visualizer now has a search box that lets you jump straight to a model by name.

Open the visualizer for a controller and type part of a model's name in the search box. The view moves to that model so you can see where it is assigned without scanning every port by hand.

On controllers with many ports and a long model list this makes it much easier to confirm where a particular prop is wired, or to find a model you need to move to a different port.
