Multi-Model xmodel Files

You can now export and import several models together in a single xmodel file.

Select the models you want, export them, and they are saved as one file. When you import that file back, all of the models come in together. Start channel references between the models are preserved rather than being reset.

This is handy for sharing a whole prop or a matched set of models with someone else, or for moving a group of related models between shows without exporting each one separately.
