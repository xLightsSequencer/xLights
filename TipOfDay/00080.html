Custom Color Picker

xLights includes an experimental custom color picker that you can turn on under Preferences, in the Other section.

Once enabled, picking a color gives you an HSV color disc, a separate brightness slider, swatches for the standard CSS named colors, a list of recently used colors, and a box for typing a hex value directly. It is designed to make finding an exact shade quicker than hunting through the default picker.

If you choose a lot of custom colors while sequencing, the recent colors list and hex entry alone can save a fair amount of clicking.
