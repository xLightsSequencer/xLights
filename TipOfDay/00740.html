Virtual Matrix On FPP

A Falcon Player instance can drive a screen, projector, or HDMI panel as a "Virtual Matrix" instead of a string of pixels. xLights supports this as a controller type so you can sequence to it like any other display.

Set the controller type to Virtual Matrix on the Controllers tab, and a Discover will also pick up FPP instances already configured to run as one. From there you map a matrix model onto it and upload as usual.

It's a handy way to add a video wall or monitor to your show without buying pixel hardware, reusing a spare Raspberry Pi or BeagleBone you already have running FPP.
