Model Icons In The Import Dialog

When you import effects from another sequence, xLights can show a small icon for each model if it finds an rgbeffects file to read them from.

These thumbnails appear next to the model names in the import mapping dialog, giving you a quick visual reminder of what each model is. That makes it far easier to match the incoming models to the right targets in your own layout.

For a big show with lots of similarly named props, the icons cut down on mistakes and let you map effects faster than reading names alone.
