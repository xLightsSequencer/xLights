Horizontal Tree Model

The Tree model can be laid out horizontally, not just as the usual upright cone.

Choose the horizontal option in the tree model's settings and the strands run sideways instead of rising to a point. This lets a tree model stand in for props that are wide rather than tall, while still giving you the familiar tree wiring and controls.

It is a quick way to reuse the tree model for a different shape without building something custom, which keeps your layout simpler to manage.
