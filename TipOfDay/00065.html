Lyric User Dictionary

When xLights breaks down lyrics into phonemes it looks up each word in its built-in pronouncing dictionary. If a word is not found — a name, a place, slang, or anything unusual — xLights simply skips it and produces no mouth shapes for that word.

You can fix this with the User Dictionary. Go to Tools > User Dictionary and add the missing word along with how it should be pronounced. The user dictionary is loaded last, so it also lets you correct any existing pronunciation you disagree with.

User dictionary entries are stored in your show folder, so they travel with your sequence when you share or back it up.
