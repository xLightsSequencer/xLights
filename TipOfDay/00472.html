Define Zigzag Wiring For Upload

You can describe a model's zigzag wiring in xLights and have that information included when you upload to controllers like Falcon and FPP.

Set the zigzag on the model's controller connection, and it is also saved with the model when you export and import an xmodel file. This means the controller knows how the string actually folds back on itself, so the pixels light in the order the model expects.

Defining it once in xLights saves you from configuring the same zigzag by hand on the controller, and keeps the wiring detail with the model if you share or reuse it.
