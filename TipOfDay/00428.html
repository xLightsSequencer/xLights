Filter VU Meter Timing Track Triggers

When a VU Meter effect reacts to a timing track, you can add a text filter so it only responds to certain timing marks rather than every one.

Set the VU Meter to use a timing track as its source, then enter text in the filter box. Only timing marks whose label matches your filter will trigger the meter.

This lets one timing track drive several different reactions. By labeling your marks and filtering on those labels, you can make one effect fire on the chorus, another on a particular word, and so on, all from the same track.
