Filter Models With Regex When Building Groups

When adding models to a group, you can now filter the model list using a regular expression instead of scrolling through everything.

Type a pattern into the filter box and the list narrows to just the matching models. For example you can match all of your matrix models or every model with a common prefix, then add the whole filtered set in one go.

This is a big time saver on large layouts with hundreds of models, where finding the right ones by eye is slow and easy to get wrong.
