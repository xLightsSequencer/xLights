Full xLights Control On FPP Controllers

FPP based controllers offer a "Full xLights Control" option, which lets xLights manage the controller's output configuration completely. When enabled, xLights defines the ports and outputs rather than leaving them set up in FPP.

You turn this on in the controller settings for the FPP device. From then on, uploading from xLights writes the full output config, keeping the controller in step with your layout.

It is the cleanest way to run an FPP cape or hat when xLights is your source of truth, because there is only one place defining where each pixel lives.
