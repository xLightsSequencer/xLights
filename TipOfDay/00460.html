Tile And Duplicate Effects

You can take an effect you have already placed and repeat it across a stretch of the timeline in one step. Right-click the effect and choose the option to duplicate or tile it.

xLights copies the effect end to end so it fills the range you want, which saves you from dragging out the same effect over and over by hand. It is handy for building a repeating pulse, chase, or background pattern that runs for a whole section of a song.

Because every copy carries the same settings, you can perfect one effect and then tile it, knowing each repeat will look exactly like the first.
