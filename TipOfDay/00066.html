Lua Script: Render and Export Model to Video

The built-in Lua script RenderExportModel (found under Tools > Run Script) lets you render one or more sequences and export a single model as a video file — all in one step, without manually opening each sequence.

When you run the script you are prompted to choose:

  • The model to export
  • The video format (see below)
  • Which sequences to process
  • Whether to force high-definition rendering

Each sequence is opened, rendered, exported to a sub-folder named after the model, then closed automatically.

Available video formats

Format choiceOutputNotes
Compressed MP4.mp4H.264 — smallest file, good compatibility
High Quality MP4.mp4HEVC — near-lossless, smaller than raw
Uncompressed AVI.aviRaw RGB — largest file, no quality loss
Lossless RGB MOV.movLossless, Mac-friendly alternative to AVI
ProRes 4444 MOV.mov4:4:4 near-lossless; ideal for video editing
HD ProRes MOV.movProRes 4444 scaled to 1920×1080

The BatchRenderExportModel script works the same way but processes every model in a group automatically.
