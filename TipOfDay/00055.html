Command Palette

Press Ctrl+Shift+K (Cmd+Shift+K on a Mac) and a searchable launcher pops up over the sequencer.

Start typing and it fuzzy matches against every menu command and every effect, so you don't need to remember which submenu something lives in. Use the arrow keys to move through the results and Enter to run whatever is highlighted.

On macOS the results even show the SF Symbol icon for each command, which makes it quicker to spot the one you want.

If you keep forgetting where a particular command lives in the menus, this is faster than hunting for it every time.
