Import Submodels From A CSV File

The submodels dialog can import submodel definitions from a CSV file, so you can lay them out in a spreadsheet and bring them in all at once.

Open the submodels dialog for a model and look for the CSV import option. Each row in the file describes a submodel and the nodes it contains, which xLights reads and turns into real submodels on the model.

This is a big time saver for complex props with many submodels, since editing rows in a spreadsheet is far quicker than creating each submodel by hand, and the file is easy to keep and reuse.
