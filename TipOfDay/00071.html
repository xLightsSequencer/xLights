Per Model Default For Groups

Model groups now have a "Per Model Default" layout option in their properties.

When this is set, any effect you drop on the group that uses the Default render style will automatically render per model rather than across the whole group buffer. You set it once on the group and every Default effect on that group follows along.

It is handy when a group is made up of identical props, like a row of mini trees or matching arches, and you want an effect to play the same way on each one instead of being stretched across all of them.
