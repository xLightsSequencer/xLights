KiNet Output Support

xLights can output using the KiNet protocol, used by some Color Kinetics and compatible lighting hardware.

When you add a controller on the setup tab, KiNet is available as one of the output protocols you can choose. Configure it like any other Ethernet output by setting the controller IP and the channels it should carry.

If you have gear that speaks KiNet, this lets you drive it from the same sequence as the rest of your display. There is no need for a separate program just to handle those fixtures.
