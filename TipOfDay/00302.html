Copy A Model With Its Submodels

On the Layout tab you can copy a model and bring all of its submodels along with it. Right-click the model and choose the copy option that includes submodels.

The new copy arrives with the same submodel definitions already set up, so you don't have to recreate every submodel by hand on the duplicate.

This is a big time saver when you have several identical props, such as a row of matching trees or arches, where you built careful submodels on the first one and want the rest to share them.
