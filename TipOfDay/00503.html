Text Effect Replacement Tokens

The Text effect can pull information straight from your song instead of you typing it in. Use the tokens ${TITLE}, ${ARTIST}, ${ALBUM}, and ${FILENAME} anywhere in the text.

When the effect renders, each token is replaced with the matching detail from the audio file loaded in your sequence. For example, typing Now Playing: ${TITLE} shows the real song title.

This is handy for jukebox-style displays and reusable templates, because the same effect shows the right words no matter which song the sequence is built around.
