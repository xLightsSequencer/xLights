RGB Image Models

The Image model can use full RGB color from your source picture. When you set up an Image model on the Layout tab, you can have it pick up the colors from the image rather than treating it as a single-color shape.

This lets a static image act as part of your display with its original colors, which is handy for signs, logos, or decorative panels that should always look the same.

It is an easy way to add a fixed, colorful element to your show without building a custom model or sequencing it frame by frame.
