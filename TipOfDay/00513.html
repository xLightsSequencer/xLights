Keybindings For Faces States And Submodels

You can set up keyboard shortcuts that drop preset Faces, States, Submodel, and model-data effects straight onto the grid.

Define these in the keybindings editor the same way you set up shortcuts for ordinary effects. Each one remembers the settings you captured, so pressing the key lays down a fully configured effect.

If you build a lot of singing-face or state sequences, this turns a repetitive multi-step drag into a single keystroke and keeps your work consistent from song to song.
