Bulk Edit State Effects

The State choice on a State effect can now be bulk edited, so you can change it across many effects at once.

Select all the State effects you want to update, then change the state selection and apply it to the whole selection rather than editing each effect one at a time. This works the same way as bulk editing other effect settings on the grid.

If you reuse the same State model across a sequence and decide a different state looks better, this saves a lot of repetitive clicking and keeps all those effects consistent.
