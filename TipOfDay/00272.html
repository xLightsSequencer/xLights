More Strings For Custom Models

Custom models can now use more than 48 strings.

When you build or edit a custom model, the string count is no longer capped at the old limit, so large props that need many separate strings can be defined as a single model. You'll see the higher limit when setting up the number of strings in the custom model editor.

This makes it easier to represent big, complex elements as one model instead of splitting them across several, which keeps your layout and sequencing simpler to manage.
