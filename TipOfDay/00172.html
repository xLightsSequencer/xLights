Divide Timings By Interval

There are keyboard shortcuts for splitting a timing mark into a set number of equal pieces using predefined intervals.

Select a timing section on a track and press the shortcut for the division you want, and xLights drops in the extra marks to break that span into halves, thirds, quarters, and so on. You can check and customize these shortcuts in the keyboard bindings settings.

Because it is all on the keyboard, you can subdivide as you go without reaching for a menu, which keeps you moving while you lay out the fine timing detail of a song.
