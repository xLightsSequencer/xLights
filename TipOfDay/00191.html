More Pixels Per Port On WLED ESP32

WLED controllers running on an ESP32 can now be configured for up to 1200 pixels per port.

When you set up a WLED ESP32 controller in the Controllers tab, you can assign more pixels to a single output than before, so a long run no longer has to be split across multiple ports.

This makes it easier to drive larger props and longer pixel runs from one port, simplifying both your wiring and your model setup.
