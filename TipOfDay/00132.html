Effect Sequences For Reusable Overlays

The New Sequence wizard now offers an "Effect Sequence" type alongside the usual musical and animation sequences.

Effect sequences are built to be reusable overlays. They save to sparse FSEQ files that contain only the exact channel ranges you used, skip the default timing tracks, and skip view selection. Pick the Effect Sequence type when you start a new sequence to use it.

Because they only touch the channels you actually sequenced, these overlays can be layered on top of your show without stepping on the other models already playing on your controllers.
