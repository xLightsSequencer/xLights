An Alignment Grid On The Layout

The 2D Layout view can show an optional alignment grid behind your models. It gives you evenly spaced reference lines to position props against.

Turn it on from the Layout tab controls and use the lines to line models up in neat rows and columns, or to keep matching props the same distance apart. It makes it much easier to judge spacing by eye than working against a blank background.

The grid is only a visual aid in the editor, so it never affects your output. It is purely there to help you lay things out cleanly.
