Multiple Strings On A PolyLine

A PolyLine model can hold more than one string, so a single model can follow a path that is actually made up of several separate pixel runs.

Set the number of strings in the PolyLine's properties and tell it where each string starts. xLights then tracks the wiring breaks along the path for you.

This lets you describe runs like a roofline or a fence that wraps around your house as one tidy model, instead of stitching together a handful of small models to cover the same shape.
