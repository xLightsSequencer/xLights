Import Submodels From Another Layout

In the Submodels dialog you can import submodels that you already defined in another layout. Open the dialog for a model and choose the option to import submodels from another layout file.

If you carefully built submodels on a similar prop in a different show, you can pull those definitions straight in instead of recreating each one node by node.

This is especially useful when you reuse the same physical props across multiple shows, since the submodel work you did once can follow you to every layout.
