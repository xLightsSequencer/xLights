Control The Render Cache Size

xLights can cache rendered effects so that replaying or re-rendering parts of a sequence is much faster. You can set a limit on how large that cache is allowed to grow in Preferences.

Look in the rendering settings of Preferences for the maximum render cache size. Raising it lets xLights keep more rendered work on hand, which speeds up editing; lowering it frees disk space if you are tight on room.

It is a handy knob to tune to your own machine. If you have plenty of fast storage, a larger cache makes a busy sequence feel snappier as you scrub back and forth.
