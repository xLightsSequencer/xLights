Stretch Custom Model Data To The Grid

When editing a custom model, xLights can stretch your existing node data to fill a new grid size. Change the grid dimensions and the nodes you have placed are scaled to fit instead of staying bunched in one corner.

This saves a lot of rework when you realise a custom model needs more rows or columns for finer detail. Rather than re-placing every node by hand, you let xLights spread the layout across the larger grid.

It is a real help for big or intricate custom props, where redoing the node positions from scratch would otherwise take a long time.
