Map One Source To Many Models

In the import mapping dialog you can select several of your models, submodels or groups at once and map a single available source onto all of them in one go.

Highlight the target rows you want, then assign the source. Every selected row picks up the same mapping instead of you repeating the assignment one row at a time.

This is a big help on large imports where many props should all receive the same incoming effects, turning a long click-by-click chore into a single action.
