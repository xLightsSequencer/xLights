Node Count In Model Tooltips

Hover over a model in the layout and its tooltip includes the node count for that model.

You do not have to open the model's properties to find out how many nodes it has. Just rest the pointer on it and the popup tells you, alongside the other details it shows.

This is a fast way to sanity check a model while you work. If a prop's node count looks wrong, you will spot it right away instead of discovering it later when the wiring does not line up.
