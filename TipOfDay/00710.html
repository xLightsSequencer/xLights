Multiple Media Directories

Besides your show directory, xLights lets you set up several additional media and resource directories. Files found in any of them are treated as part of your show.

This means you can keep a shared library of music, images, videos or meshes in one place and reuse it across shows, rather than copying the same files into every show folder.

When xLights resolves a file path it searches these directories, so a sequence built against a shared library keeps finding its media even though the files live outside the show folder.
