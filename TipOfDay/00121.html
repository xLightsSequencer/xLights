Face Definition Previews

When you set up a matrix face definition, xLights now shows a preview of each mouth and eye image as you define it.

You will see the previews in the Face Definition dialog where you assign the pictures for each phoneme and expression. Each entry shows the image it points at, so you can confirm it is the one you meant.

This makes building a singing face much less error prone. Instead of matching file names in your head, you can glance at the previews and immediately spot a picture that is wrong, missing, or in the wrong slot.
