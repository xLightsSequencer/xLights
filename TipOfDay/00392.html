Import Twinkly As A 3D Model

If you have a Twinkly string that has been mapped in their app, xLights can import that mapping as a 3D model.

Twinkly's app records where each light sits in space. Bringing that data into xLights builds a model with the lights placed in three dimensions, so effects wrap around the real shape of how you hung the string rather than treating it as a flat line.

This saves you from trying to recreate an irregular layout by hand. The captured positions come straight across, giving you a model that matches your actual installation.
