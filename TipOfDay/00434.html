Metronome Timing With Tags

A metronome timing track can label its marks with tags, giving each evenly spaced beat a name you can react to.

Create the metronome timing track and enable its tags, then those labels travel with the marks. Effects that read a timing track, such as the State effect, can use the tags to know which beat they are on.

This is great for driving counting or stepping behavior, like advancing a state or cycling through a set of looks one tag at a time, all locked to a steady tempo.
