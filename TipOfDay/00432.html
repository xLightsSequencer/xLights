Dark Mode On Windows

On Windows, xLights can display its interface in a dark theme with light text on dark backgrounds.

You turn it on in the preferences. If you would rather keep the standard light interface, there is also an option to suppress dark mode so xLights stays light even when Windows itself is set to dark.

A dark interface is easier on the eyes during long sequencing sessions, especially when you are working at night near your display, so it is worth trying whichever way you prefer.
