Save A Default Layout View

On the Layout tab you can save a default camera view, the angle and zoom you like to look at your display from. Once saved, xLights returns to that view when it opens.

This is handy in 3D mode or with a large layout, where the preview might otherwise start zoomed out or pointing somewhere awkward. Set up the framing you want, then save it as the default.

Every time you come back to the Layout tab it greets you with your preferred perspective, so you are not constantly panning and zooming to get back to where you work.
