Import Model Groups From Another Sequence

If you've built up handy model groups in one show, you can pull them into another rather than recreating each one by hand.

xLights can import model groups from another RGB effects file, bringing the group definitions across so the same combinations of models are ready to use in your current layout.

This saves real time when you reuse a layout or share groupings between shows, since carefully assembled groups carry over in one step instead of being rebuilt model by model.
