Imports Remember Your Last Folder

When you import a file, xLights now remembers the folder you used last time and opens the file browser there again.

You do not have to do anything to turn this on. The next time you import, the dialog starts in the same directory you browsed to before instead of sending you back to a default location.

If you keep your sequences, models, or media tucked away in a folder several levels deep, this saves you from clicking through the same path over and over each time you bring something in.
