Repeating A Custom Value Curve With Cycles

Custom value curves now have a Cycles parameter, settable from 1 to 10.

Instead of drawing the same up-and-down shape over and over across a long effect, draw it once and set Cycles to however many times you want it repeated across the buffer. xLights replicates the shape for you.

This is a nice time saver for anything rhythmic, like a curve that should pulse in time with a repeating beat, where redrawing the same shape by hand for every repetition would be tedious and hard to keep consistent.
