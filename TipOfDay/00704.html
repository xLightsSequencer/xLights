Timing Track Value Curves

Some value curves can read from a timing track, letting the marks you have placed drive how a setting changes over time instead of using a fixed mathematical shape.

When you edit a value curve, look for the timing-track based options. The curve then steps or ramps in time with the beats and sections defined on your chosen timing track.

This is a powerful way to tie effect parameters to the music. Brightness, speed or position can all move on the beat without you keying each change in by hand.
