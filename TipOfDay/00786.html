Import GDTF Fixtures

xLights can read GDTF files, a standard format used to describe DMX lighting fixtures such as moving heads and wash lights. When you import a model you can point it at a GDTF file to set up the fixture.

The GDTF file carries the fixture's channel layout and capabilities, so xLights can configure the DMX channels for you instead of you entering them by hand. Many manufacturers publish GDTF files for their products.

This makes adding a professional fixture far less error prone, since the channel mapping comes straight from the maker's own definition rather than from a manual you have to translate yourself.
