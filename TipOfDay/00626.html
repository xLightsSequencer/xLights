Create A State From A DMX Effect

When you have a DMX fixture posed exactly how you like it with a DMX effect, you can turn that pose into a saved state definition rather than rebuilding it by hand.

From the DMX effect you can capture its current channel values straight into a state on the model. That state then shows up in the State effect like any other.

This is a real time saver for moving heads and other DMX fixtures, letting you dial in a look once and then recall it again and again throughout your sequence.
