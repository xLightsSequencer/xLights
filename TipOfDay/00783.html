Export An Effect As A GIF

You can save what a model is doing as an animated GIF image. Use the export model option and choose the GIF format to produce a small looping animation of your effect.

It captures the model exactly as it renders in the preview, so the GIF shows the real motion and colours rather than a static picture. You can limit it to the part of the timeline you care about so the file stays small.

GIFs are handy for sharing a quick preview of an effect with others, posting on forums, or keeping a visual library of looks you like without having to open xLights to remember them.
