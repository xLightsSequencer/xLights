Pinpoint Errors In Your Effects File

If your rgbeffects file has a problem in its XML, xLights now shows you the error and the exact location where it was found.

Instead of a generic failure to load, you get the specific spot in the file that is malformed, which makes it far easier to track down what went wrong, whether the file was hand-edited or got corrupted.

This is a real time saver when recovering a layout that will not open, since it points you straight at the line that needs fixing.
