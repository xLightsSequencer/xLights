Generate A Color Palette With AI

xLights can build a color palette for you using AI. Instead of picking each color by hand, you describe the look you want and let it suggest a set of colors that work together.

Look for the Generate Palette option in the color panel where you choose your palette colors. Type in a description, such as a holiday theme, a mood, or a scene, and it returns a palette you can drop straight into the effect you are editing.

It is a quick way to get a coordinated set of colors when you are not sure where to start, and a handy source of inspiration when you want something different from your usual go-to colors.
