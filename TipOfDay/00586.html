Import Views From Other Sequences

Views let you group a chosen set of models together so you can focus on just those rows in the sequencer. Rather than rebuilding the same views in every show, you can import views from another sequence file or from your layout file.

Look for the option to import views when you are managing your views. Pick the file to pull them from, and the views you have already set up come straight across.

This is a real time saver if you reuse the same groupings season after season, since you can carry your carefully arranged views from one show to the next.
