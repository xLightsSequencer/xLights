FLAC Audio Support

xLights can use FLAC audio files for your sequences, alongside the usual MP3 and other formats. Just pick a FLAC file when you create or set the audio for a sequence.

FLAC is a lossless format, so the audio keeps full quality rather than being compressed down like an MP3. That can mean cleaner beat detection and a better reference while you sequence.

If you have your music in FLAC already there is no need to convert it first. Point xLights at the file and it works like any other supported audio.
