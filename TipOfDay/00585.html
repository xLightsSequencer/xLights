Buffer Styles On Submodels

Submodels can use render buffer styles, just like full models. When you put an effect on a submodel you can choose how its buffer is arranged, rather than being stuck with the default layout.

You set this in the buffer style option on the effect, the same place you would for a regular model. Different styles change how an effect maps across the submodel's nodes, which can dramatically change how it looks.

This gives you far more control over effects on a portion of a prop, so a submodel can flow in the direction you want instead of however the nodes happen to be ordered.
