Improved Check Sequence Report

The Check Sequence report is now presented as an easy to read HTML page.

Run Check Sequence from the Tools menu and the results open in a clearer, better organized layout that's simpler to scroll through and navigate. Warnings and errors are grouped so you can spot the items that need attention more quickly.

It's a good habit to run Check Sequence before a show or before uploading, and the cleaner report makes it much less of a chore to read through what it finds.
