WLED Controllers Are Now Auto-Discovered

If you have WLED devices on your network, xLights will now find them for you using mDNS, the same way it discovers other controllers.

Run a Discover from the Controllers tab and any WLED controllers it finds get added automatically as DDP outputs, with the channel size and layout already worked out for you.

This saves manually adding the controller, picking DDP, and figuring out the pixel count yourself. It's still worth double-checking the auto-detected layout matches what you actually wired up.
