Use Vendor Supplied Models When Importing

When you import a layout and one of the models matches a model that the vendor has supplied, xLights offers to use the vendor's version instead. It recognises the match and prompts you so you can choose which one to keep.

The vendor supplied model usually has the correct node counts, wiring order and a proper preview image already set up, so it tends to be more accurate than a hand built copy. Just accept the prompt when it appears during the import.

This saves you from rebuilding a prop from scratch and reduces the chance of getting node order or counts wrong, which would otherwise show up as scrambled effects later.
