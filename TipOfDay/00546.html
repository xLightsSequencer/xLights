Bulk Set Controller Ports On The Layout

The Layout tab can apply controller settings to several models at once. Select a group of models and you can set just the protocol, set the controller port number, and even increment the port number automatically for each model.

The auto-increment option is the real time saver. Select your props in order, set the first port, and let xLights number the rest, so a long string of models lands on consecutive ports in one step.

This turns what used to be a tedious model-by-model job into a quick bulk edit, which is ideal when you are wiring up a new controller full of similar props.
