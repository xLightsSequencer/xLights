Base Show Folders

A show folder can inherit its controllers and models from a separate base show folder.

You set up the shared layout and controller setup once in the base folder, and other show folders build on top of it. Models and controllers defined in the base appear in the show that inherits from it, while each show can still add its own pieces.

This is powerful when you run more than one show off the same physical display, or want a common foundation across seasons. Update the base once and every show that inherits from it picks up the change.
