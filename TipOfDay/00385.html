Make A Block Of Ports A Dumb Receiver

In the controller visualizer there is a quick way to turn a whole block of ports into a dumb receiver in one step.

Rather than setting each port's receiver type one at a time, select the range and apply the dumb receiver setting to all of them at once. The visualizer updates the block together.

If you run smart receivers alongside dumb ones, this saves a lot of repetitive clicking when wiring out a larger controller. Set the block in one action and move on.
